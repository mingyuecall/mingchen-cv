# CHEN Ming (Eric) Academic CV

This repository contains a static GitHub Pages CV site built from the source Word document.

## Publish on GitHub Pages

1. Create a GitHub repository named `ericmingchen.github.io` or any repository name you prefer.
2. Upload `index.html`, `styles.css`, and the source CV document.
3. In GitHub, open `Settings` > `Pages`.
4. Choose `Deploy from a branch`, select the `main` branch, and use the repository root.
5. Save the settings. GitHub will publish the page after the first deployment finishes.

## Edit Content

Most CV content is in `index.html`. Visual styling is in `styles.css`.
